Accessibility Statement

This website aims to achieve and maintain WCAG 2.1 AA standards in accordance with the UK law and legislation.

I want as many people as possible to be able to use this website. In the pages of this website I attempted to follow W3C (Worldwide Web Consortium) guidelines. For example, on these web pages you should be able to:

Use screen readers without any major problems - alt properties for images are included where needed and formed in such a way that they should not disadvantage disabled users;

Distinguish between elements easily - all elements on the page are placed with care and in such a way that any movement issues should not impact the navigation of the website;

Apply your own styles if needed - separate style sheets are used for visual layout.

This is not an official site of the University of Sheffield.

All views and statements are an opinion of the creator of the site, which is a Computer Science student.
